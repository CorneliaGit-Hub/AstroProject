Félicitations, vous avez téléchargé avec succès le kit @font-face !

Ce kit vous est fourni par Fonts2u.com – le plus grand référentiel en ligne de polices gratuites
pour Windows et Mac.

Comment utiliser ce kit @font-face ?

Pour intégrer la police à votre site Web, procédez comme suit :

1. Téléchargez les quatre fichiers de police (.ttf, .woff, .eot, .svg) et le fichier stylesheet.css sur
votre site Web à l'aide du panneau de contrôle de votre compte d'hébergement ou via ftp. Tous
ces fichiers sont disponibles dans le répertoire racine .zip de fontface.

2. Déclarez la police choisie en copiant le code CSS généré fourni dans le
kit @font-face et collez-le dans votre fichier CSS principal, ou importez-le simplement
en tant que CSS séparé.

3. Associez la police intégrée à n'importe quel élément de votre choix en utilisant la syntaxe CSS standard (p. ex. h1 {font-family:'declared font name';}

4. Profitez de la nouvelle police sur votre site Web.
